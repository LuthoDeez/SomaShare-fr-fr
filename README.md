Project description 

We’ve developed a web-based platform called SomaShare to make it safe and easy for students to purchase, sell, and trade textbooks as well as other study-related goods. Users can create adverts, browse listings, make offers, and manage transactions through an organized system. 

Setup instructions

Applications needed 


Visual Studio 2022 

.NET SDK installed 

SQL Server Management Studio (optional) 

SQL Server / SQL Server Express 
 

How to Run the Project 

1. Clone the repository 

2. Open the solution file: 
SomaShare.sln

4. Check your connection string inside: 
appsettings.json

6. Open Package Manager Console and run: 
update-database

8. Run the application: 
Press F5 in Visual Studio

ASP.NET Core MVC 

Entity Framework Core 

SQL Server
